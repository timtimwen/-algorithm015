# 学习笔记


## 数组 Array

|  操作  | 时间复杂度 |
| :----: | :--------: |
| lookup |    O(1)    |
| insert |    O(n)    |
| delete |    O(n)    |


## 链表 Linked List

|  操作   | 时间复杂度 |
| :-----: | :--------: |
| prepend |    O(1)    |
| append  |    O(1)    |
| lookup  |    O(n)    |
| insert  |    O(1)    |
| delete  |    O(1)    |


## 跳表 Skip List

|  操作  | 时间复杂度 |
| :----: | :--------: |
| lookup | O(log(n))  |
| insert | O(log(n))  |
| delete | O(log(n))  |
